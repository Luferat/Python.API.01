# Python.API.01

Minha primeira API REST em Python.

## Requisitos

- Python 3.10^
- Flask

## Opcionais

- [DB Browser (SQLite)](https://sqlitebrowser.org/) para gerenciar o banco de dados local.

## Rotas e Endpoints

Obtém todos os registros válidos de 'item'.
 - Request method → GET
 - Request endpoint → /items
 - Response → JSON
